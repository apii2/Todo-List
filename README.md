# Todo List

This is a Todo List application built with Next.js and TypeScript. It allows users to manage their tasks efficiently with a clean and modern interface.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Responsive design
- Theme support

## Prerequisites
To run this project, you need to have the following installed:
- Node.js (v16 or higher)
- npm or yarn

## Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/Todo-List.git
cd Todo-List
```

### 2. Install Dependencies
```bash
npm install
# or
yarn install
```

### 3. Set Up the Backend
This project requires a backend server to manage tasks. Clone and run the backend server from the following repository:

[Todo List API](https://github.com/apii2/Todo-List-api)

Follow the instructions in the backend repository to set it up and run the server locally.

### 4. Run the Application
Start the development server:
```bash
npm run dev
# or
yarn dev
```

Open your browser and navigate to `http://localhost:3000` to see the application.
